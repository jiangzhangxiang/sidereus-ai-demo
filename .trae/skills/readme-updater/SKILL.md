# README Updater

这个技能能够根据项目的变化自动更新 README.md 文件的内容，确保文档与项目保持同步。

## 功能特性

- **所有文档输出都强制使用中文**：确保生成的文档内容都是中文
- **自动检测项目结构变化**：监控项目目录、文件和配置的变化
- **智能内容更新**：根据技术栈、依赖、路由等信息生成相应的文档
- **格式化输出**：保持一致的 Markdown 格式和文档风格
- **支持多种项目类型**：适配 Monorepo、全栈应用等多种架构

## 使用场景

- 当用户要求更新 README.md 文件时
- 项目新增功能或模块时
- 技术栈或依赖发生变化时
- 项目结构重组时

## 实现逻辑

### 1. 项目分析
- 扫描项目根目录结构
- 识别技术栈和框架
- 分析路由、组件、服务等关键元素
- 检查配置文件和依赖信息

### 2. 文档生成
- 根据项目结构生成目录
- 创建功能模块说明
- 更新技术栈介绍
- 添加使用说明和部署指南

### 3. 内容更新策略
- 保持现有内容的完整性
- 增量更新新添加的内容
- 移除过时的信息
- 保持文档的逻辑结构

## 技术实现

- 使用 Node.js 文件系统 API 进行文件操作
- 通过正则表达式匹配和替换更新内容
- 支持多文件内容的合并与同步
- 提供回滚机制确保文档安全性

## 使用示例

```bash
# 自动更新 README.md
readme-updater --auto

# 查看项目结构分析
readme-updater --analyze

# 预览更改内容
readme-updater --preview
```
